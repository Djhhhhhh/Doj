# 前言

​	现大三上，本项目是本人第一次完成的大型项目，为保证项目完整性，故写此篇readme文档。

​	该项目已采用为 **哈尔滨理工大学（威海）算法竞赛实验室** 项目。

# 一、总概

-   基于Vue和SpringBoot、SpringCloudAlibaba构建的前后端分离，分布式架构的评测系统
-   目前仅支持Cpp和Java语言评测
